# Token Disambiguation

<Aside type="danger">
**SSID (Self-Sovereign Identity) is NOT affiliated with, endorsed by, or related to any token trading on Binance Smart Chain (BSC) or any other exchange under the name "SSID Token".**
</Aside>

## What is Official?

| Official | Not Official |
|----------|-------------|
| This documentation site | Any token on BSC/DEX with "SSID" ticker |
| [SSID-open-core](https://github.com/EduBrainBoost/SSID-open-core) repository | Any social media account not linked from official repos |
| SSID utility token (if/when launched through official governance) | Any airdrop, presale, or investment scheme using "SSID" name |

## How to Verify

1. **Check the source** — Official information comes from the GitHub repositories listed above
2. **Check governance records** — Any official token launch will have evidence in the WORM audit log
3. **No presale** — SSID does not and will never conduct a presale or ICO
4. **No promises of returns** — The SSID token is utility-only

## Why the Name Collision?

"SSID" is a common acronym (also used for WiFi network names). Name collisions in the crypto space are frequent and often intentional by scam projects. This disambiguation page exists to protect users.

## Report Fraud

If you encounter a project falsely claiming to be affiliated with SSID:

1. Do not interact with it
2. Report it on the relevant exchange/platform
3. Notify the SSID team through official GitHub channels

<Aside type="caution">
Always verify through official channels before interacting with any token or project claiming to be SSID.
</Aside>

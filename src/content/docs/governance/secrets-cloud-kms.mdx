## Overview

For production deployments, SSID integrates with cloud-native key management services backed by Hardware Security Modules (HSMs).

## AWS KMS

### Setup

```bash
# Create KMS key for evidence encryption
aws kms create-key \
  --description "SSID Evidence Encryption" \
  --key-usage ENCRYPT_DECRYPT \
  --origin AWS_KMS

# Create alias
aws kms create-alias \
  --alias-name alias/ssid-evidence \
  --target-key-id <key-id>
```

### Python Integration

```python
import boto3

kms = boto3.client("kms")

def encrypt_evidence(plaintext: bytes) -> bytes:
    response = kms.encrypt(
        KeyId="alias/ssid-evidence",
        Plaintext=plaintext,
        EncryptionContext={"service": "ssid-ems", "type": "evidence"},
    )
    return response["CiphertextBlob"]

def decrypt_evidence(ciphertext: bytes) -> bytes:
    response = kms.decrypt(
        CiphertextBlob=ciphertext,
        EncryptionContext={"service": "ssid-ems", "type": "evidence"},
    )
    return response["Plaintext"]
```

### Audit

All KMS operations are logged in AWS CloudTrail automatically.

## Azure Key Vault

### Setup

```bash
# Create Key Vault
az keyvault create \
  --name ssid-keyvault \
  --resource-group ssid-rg \
  --sku premium  # HSM-backed

# Create key
az keyvault key create \
  --vault-name ssid-keyvault \
  --name ssid-evidence \
  --kty RSA-HSM \
  --size 4096
```

### Python Integration

```python
from azure.identity import DefaultAzureCredential
from azure.keyvault.keys.crypto import CryptographyClient, EncryptionAlgorithm

credential = DefaultAzureCredential()
client = CryptographyClient(
    "https://ssid-keyvault.vault.azure.net/keys/ssid-evidence",
    credential,
)

result = client.encrypt(EncryptionAlgorithm.rsa_oaep_256, plaintext)
```

## FIPS 140-2 Compliance

| Provider | Level | Validation |
|----------|-------|-----------|
| AWS KMS | Level 2 (default), Level 3 (CloudHSM) | NIST validated |
| Azure Key Vault Premium | Level 2 (software), Level 3 (HSM) | NIST validated |
| HashiCorp Vault Enterprise | Level 2 (Seal), Level 3 (with HSM auto-unseal) | Partner validated |

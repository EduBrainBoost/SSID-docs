## Network Configuration

### Hardhat config for testnets

```javascript
module.exports = {
  networks: {
    sepolia: {
      url: process.env.SEPOLIA_RPC_URL,
      accounts: [process.env.SEPOLIA_PRIVATE_KEY],
      chainId: 11155111,
    },
    mumbai: {
      url: process.env.MUMBAI_RPC_URL,
      accounts: [process.env.MUMBAI_PRIVATE_KEY],
      chainId: 80001,
    },
  },
};
```

### Web3 Provider Setup (Node.js)

```javascript
const { ethers } = require('ethers');

const sepolia = new ethers.providers.JsonRpcProvider(
  'https://sepolia.infura.io/v3/YOUR_KEY'
);

const mumbai = new ethers.providers.JsonRpcProvider(
  'https://mumbai.infura.io/v3/YOUR_KEY'
);

// Get contract instance
const contract = new ethers.Contract(
  '0xcontractaddress',
  IdentityResolverABI,
  sepolia
);
```

### Ethers.js Web (Browser)

```javascript
const provider = new ethers.providers.Web3Provider(window.ethereum);

// User connects wallet (MetaMask)
await provider.send('eth_requestAccounts', []);

// Get signer for transactions
const signer = provider.getSigner();

const contract = new ethers.Contract(
  '0xcontractaddress',
  IdentityResolverABI,
  signer
);

// Execute transaction
const tx = await contract.registerDID(did, docHash, keyMaterial);
await tx.wait();
```

## Health Check Scripts

### Verify RPC Connectivity

```bash
# Sepolia
curl -X POST https://sepolia.infura.io/v3/YOUR_KEY \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_blockNumber",
    "params": [],
    "id": 1
  }'

# Expected: {"jsonrpc":"2.0","result":"0x...","id":1}

# Mumbai
curl -X POST https://mumbai.infura.io/v3/YOUR_KEY \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",

    "params": [],
    "id": 1
  }'
```

```bash

```

## Migration from Testnet to Mainnet

When ready to deploy on mainnet:

1. **Replace RPC endpoints** with mainnet providers (same Infura/Alchemy projects)
2. **Redeploy contracts** to mainnet chain IDs (1 for Ethereum, 137 for Polygon)
3. **Update contract addresses** in frontend and backend configuration
4. **Use mainnet wallets** with real funds (in Vault, never code)
5. **Enable mainnet audits** and security gates per [Mainnet Readiness Roadmap](./mainnet-readiness)

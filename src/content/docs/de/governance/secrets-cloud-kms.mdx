## Übersicht

Für Produktions-Deployments integriert sich SSID mit Cloud-nativen Key Management Services, die durch Hardware Security Module (HSMs) gestützt werden.

## AWS KMS

### Setup

```bash
# Erstelle KMS Key für Evidence Verschlüsselung
aws kms create-key \
  --description "SSID Evidence Encryption" \
  --key-usage ENCRYPT_DECRYPT \
  --origin AWS_KMS

# Erstelle Alias
aws kms create-alias \
  --alias-name alias/ssid-evidence \
  --target-key-id <key-id>
```

### Python Integration

```python
import boto3

kms = boto3.client("kms")

def encrypt_evidence(plaintext: bytes) -> bytes:
    response = kms.encrypt(
        KeyId="alias/ssid-evidence",
        Plaintext=plaintext,
        EncryptionContext={"service": "ssid-ems", "type": "evidence"},
    )
    return response["CiphertextBlob"]

def decrypt_evidence(ciphertext: bytes) -> bytes:
    response = kms.decrypt(
        CiphertextBlob=ciphertext,
        EncryptionContext={"service": "ssid-ems", "type": "evidence"},
    )
    return response["Plaintext"]
```

### Audit

Alle KMS-Operationen werden automatisch in AWS CloudTrail protokolliert.

## Azure Key Vault

### Setup

```bash
# Erstelle Key Vault
az keyvault create \
  --name ssid-keyvault \
  --resource-group ssid-rg \
  --sku premium  # HSM-gestützt

# Erstelle Key
az keyvault key create \
  --vault-name ssid-keyvault \
  --name ssid-evidence \
  --kty RSA-HSM \
  --size 4096
```

### Python Integration

```python
from azure.identity import DefaultAzureCredential
from azure.keyvault.keys.crypto import CryptographyClient, EncryptionAlgorithm

credential = DefaultAzureCredential()
client = CryptographyClient(
    "https://ssid-keyvault.vault.azure.net/keys/ssid-evidence",
    credential,
)

result = client.encrypt(EncryptionAlgorithm.rsa_oaep_256, plaintext)
```

## FIPS 140-2 Compliance

| Provider | Level | Validierung |
|----------|-------|-----------|
| AWS KMS | Level 2 (Standard), Level 3 (CloudHSM) | NIST validiert |
| Azure Key Vault Premium | Level 2 (Software), Level 3 (HSM) | NIST validiert |
| HashiCorp Vault Enterprise | Level 2 (Seal), Level 3 (mit HSM Auto-Unseal) | Partner validiert |

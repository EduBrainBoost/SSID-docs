## Network Konfiguration

### Hardhat Config für Testnets

```javascript
module.exports = {
  networks: {
    sepolia: {
      url: process.env.SEPOLIA_RPC_URL,
      accounts: [process.env.SEPOLIA_PRIVATE_KEY],
      chainId: 11155111,
    },
    mumbai: {
      url: process.env.MUMBAI_RPC_URL,
      accounts: [process.env.MUMBAI_PRIVATE_KEY],
      chainId: 80001,
    },
  },
};
```

### Web3 Provider Setup (Node.js)

```javascript
const { ethers } = require('ethers');

const sepolia = new ethers.providers.JsonRpcProvider(
  'https://sepolia.infura.io/v3/YOUR_KEY'
);

const mumbai = new ethers.providers.JsonRpcProvider(
  'https://mumbai.infura.io/v3/YOUR_KEY'
);

// Get Contract Instanz
const contract = new ethers.Contract(
  '0xcontractaddress',
  IdentityResolverABI,
  sepolia
);
```

### Ethers.js Web (Browser)

```javascript
const provider = new ethers.providers.Web3Provider(window.ethereum);

// User verbindet Wallet (MetaMask)
await provider.send('eth_requestAccounts', []);

// Get Signer für Transaktionen
const signer = provider.getSigner();

const contract = new ethers.Contract(
  '0xcontractaddress',
  IdentityResolverABI,
  signer
);

// Execute Transaktion
const tx = await contract.registerDID(did, docHash, keyMaterial);
await tx.wait();
```

## Health Check Scripts

### Verify RPC Connectivity

```bash
# Sepolia
curl -X POST https://sepolia.infura.io/v3/YOUR_KEY \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_blockNumber",
    "params": [],
    "id": 1
  }'

# Expected: {"jsonrpc":"2.0","result":"0x...","id":1}
```

## Migration von Testnet zu Mainnet

Wenn Sie bereit sind, auf Mainnet zu deployen:

1. **Ersetzen Sie RPC Endpoints** mit Mainnet Providern (gleiche Infura/Alchemy Projekte)
2. **Redeploy Contracts** auf Mainnet Chain IDs (1 für Ethereum, 137 für Polygon)
3. **Update Contract Adressen** in Frontend und Backend Konfiguration
4. **Verwenden Sie Mainnet Wallets** mit echten Mitteln (in Vault, nie Code)
5. **Aktivieren Sie Mainnet Audits** und Security Gates per [Mainnet Readiness Roadmap](./mainnet-readiness)
